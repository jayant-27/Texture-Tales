# 🧵 Texture Tales - SQL Case Study

## 📌 Project Overview

This SQL case study analyzes sales performance for Texture Tales Clothing Company.
The objective is to generate business insights like revenue, discounts, top products, and category performance using SQL.

## 📂 Tables Used

* product_details
* sales
* product_hierarchy
* product_prices

## ❓ Business Questions Solved

* Total quantity sold
* Total revenue before discount
* Total discount amount
* Unique transactions count
* Average unique products per transaction
* Average discount per transaction
* Member vs Non-member revenue comparison
* Top 3 products by revenue
* Segment level performance
* Category level performance
* Top selling product per segment
* Top selling product per category

## 🛠️ SQL Concepts Used

* INNER JOIN
* GROUP BY
* Aggregate Functions (SUM, AVG, COUNT)
* DISTINCT
* CASE WHEN
* CTE / Subqueries
* ORDER BY & LIMIT

## 📊 Project Outcome

* Identified top performing products
* Analyzed discount impact on revenue
* Compared member vs non-member purchases
* Segment and category performance analysis

## 🚀 Tools Used

* MySQL
